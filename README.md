Spark-Ship
Open-source hybrid voice + physical + touchscreen control system for the Spark EV kit car by Trangmar Ltd
The brain of the Spark – the ultra-modular, lifetime-durable £12,500 EV kit car from Trangmar Ltd.
Hybrid Control Architecture (exactly as specified):

Offline voice mode – standard (PocketSphinx, zero internet)
Physical/manual controls – standard (heating/AC, lights, wipers, etc. – glove-friendly & failsafe)
Touchscreen UI – easy £100 option

Built for the triple-threat stack (Spark Chassis + swappable battery + Ship).
Designed from Day 1 to scale from road cars to military vehicles to CubeSats.
Quick Start (Raspberry Pi 4)
Bashgit clone https://github.com/TrangmarSpark/Spark-Ship.git
cd Spark-Ship
pip install -r requirements.txt
python main.py
Key Features

Real-time speedometer, battery SoC, range
500 W entertainment system (Bluetooth + MP3)
Multi-motor control (1–4 motors, 2WD/4WD/AWD)
Future-proof API for any sensor/actuator via the chassis conduits
Consistent UI that will later run on BAE RAD750 for space applications
